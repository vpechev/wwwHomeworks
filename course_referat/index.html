Какво е jQuery

jQuery e крос-браузърна JavaScript библиотека. Тя е създадена, за да опростява писането и обработването на динамичен HTML на клиента. Тази библиокета е най-известната Javascript библиотека в наши дни. Може да се използва безплатно в проекти с отвоен код. Синтаксисът на jQuery е мн ого лесен за научаване. Спомага навигирането и избирането на елементи от DOM. Създава анимации. Обработва събития. Също така jQuery предоставя възможността за създаване на собствени плъгини, анимации и теми, които работят със самата библиотека на ниско ниво. Позволява възможността за създаване на динамични и оптимизирани уеб страници. Поддържа интеграция с ред на брой среди за разработка. Някои сред тях са: Visual Studio, Visual Studio Code, Eclipse, Sublime Text, WebStorm

Зашо jQuery е толкова известна?

jQuery e eдна от най-известните библиотеки в наши дни. Една от причините затова е именно лесното научаване на библиотеката. Още един изключително важен факт е лесното разширяване на функционалността на библиотеката. Могат да се създават всякакви плъгини без ограничение. Просто трябва да се имплементира JavaScript функция. Използва се и се поддържа от голяма общост от програмисти.

Как да добавим jQuery към нашия код?

Инсталацията на JQuеry не изисква почти никакви усилия. Имаме различни подходи. Можем сами да хостваме сорс кода на библиотеката или да използваме алтернатива, например CDN. Aко сме избрали да е при нас, всичко, което трябва да направим е да свалим файловете на библиотеката от http://www.jquery.com и да ги добавим към нашия проект. Ако използваме външен провайдър просто трябва да копираме предоставения линк. Например: http://ajax.microsoft.com/ajax/jquery/jquery-2.1.1.min.js (CDN на Гугъл).

Селектори и манипулация на DOM

Избирането на DOM елементи много наподобява това при чистият Javascript Избиране на елементи чрез CSS селектори Например querySelectorAll $(selector) Избиране на елементи чрез име на таг $("div") //document.querySelectorAll("div"); Избиране на елементи чрез име на class $(".menu-item") //document.querySelectorAll(".menu-item"); Избиране на елементи чрез id елемент $("#navigation") Избиране на елементи чрез комбинация от различни селектори $("ul.menu li")
В повечето случаи избирането на елементи с jQuеry връща колекция от елементи дори в случаите, когато селектираният контейнер съдържа един единствен елемент. Резултатното множество може да се запази в променлива или пък да се използва още в момента на избирането. И в двата случая използваме получените елементи по един и същ начин без значение от техния брой; More at: http://learn.jquery.com/using-jquerycore/selecting-elements/

Манипулация на DOM

Подобно на чистия JavaScript, jQuery също може да манипулира DOM. Съществуват понятия като "брат" - елемент от същия ред "родител" - елемент от по-висок ред "дете" - елемент от по-нисък ред Някои примерни методи са: jQuery.next(), jQuery.prev() които връщат следващия/предишния брат съответно  Returns an HTML element  Not a [text] node
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>
var $first = $("li").first();
log($first);
//logs "Item 1"
log($first.next());
//logs "Item 2"
jQuery.parent() - Returns the parent of the element jQuery.parents(selector) - Returns the first parent that matches the selector
<div id="wrapper">
<ul id="items-list">
<li> Item 1</li>
<li> Item 2</li>
<li class="special"> Item 3</li>
<li> Item 4</li>
</ul>
</div>
var $node = $(".special");
$node.parent().attr("id");
//logs "items-list"
$node.parents("div").attr("id");
//logs "wrapper"
$node.parents("#wrapper")
.attr("id");
/logs "wrapper"
